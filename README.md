# Imposition-Simulator

A browser-based imposition simulator for printing layouts.

Site: https://yamonov.github.io/Imposition-Simulator/

## Development

```bash
npm install
npm run dev
```

## Static build

```bash
npm run build
npm run serve:docs
```

The static site is generated into `docs/`. GitHub Pages can publish it without
GitHub Actions by setting the Pages source to `main` / `/docs`.
